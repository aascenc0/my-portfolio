# Minimalist Full Stack Developer Portfolio

A responsive, high-performance, dark-mode personal portfolio website built with clean, semantic modern web standards. 

---

## Key Features

* **Dark Mode Design:** Styled with CSS custom properties (variables) for a clean, slate-and-cyan aesthetic.
* **Interactive Project Filtering:** Vanilla JavaScript filtering allowing visitors to sort projects by category (`Frontend`, `Full Stack`).
* **Self-Contained In-Memory API Sandbox:** An offline client-side API controller that simulates RESTful HTTP operations (`GET`, `POST`, `DELETE`) with real-time JSON responses.
* **Minimal Online Footprint:** Runs completely in the browser with zero external tracking scripts, SaaS dependencies, or third-party database services.
* **Fully Responsive:** Optimized for desktop, tablet, and mobile displays.

---

## Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox & Grid), Vanilla JavaScript (ES6+)
* **Architecture Concepts:** RESTful API design, State Management, DOM Manipulation, JSON parsing
* **Tooling & Deployment:** Git, VS Code, GitHub Pages
